# 故障排除

常见问题和解决方案。

## 常见问题

### 编译错误

**问题**: 编译时出现错误

**解决方案**:
1. 确保 Rust 版本 >= 1.70
2. 运行 `cargo clean` 清理构建缓存
3. 检查依赖版本是否兼容

### 运行时错误

**问题**: 程序运行时崩溃

**解决方案**:
1. 检查日志输出
2. 启用调试模式：`RUST_LOG=debug cargo run`
3. 检查 GPU 驱动是否最新

### 性能问题

**问题**: 帧率低

**解决方案**:
1. 使用性能分析器定位瓶颈
2. 启用 GPU 驱动剔除
3. 减少绘制调用
4. 使用 LOD 系统

### 资源加载失败

**问题**: 资源无法加载

**解决方案**:
1. 检查文件路径是否正确
2. 确认文件格式支持
3. 检查资源服务器是否初始化

## 获取帮助

如果问题仍未解决：

1. 查看 [FAQ](./FAQ.md)
2. 搜索 [Issues](https://github.com/username/game_engine/issues)
3. 提交新 Issue（包含错误日志和复现步骤）
